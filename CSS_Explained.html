CSS Explained

Welcome to my CSS page. First of all, what is CSS you might ask? CSS stands for Cascading Style Sheets. Metaphorically speaking, you could say that it's the icing of the cake in terms of HTML. With CSS , you can do a lot of things. You can change the size of the the text, images, etc. You can also add colors to your page and many more! The examples are shown below. There's another concept introduced in CSS. It's called CSS Selectors. CSS selectors are basically properties that allow you to use the elements you want to style and try out. There are multiple CSS selectors but here are the main three. The first one is class,(which has a period(.) as a symbol), # which is the symbol for an id, and * which selects all the elements.
